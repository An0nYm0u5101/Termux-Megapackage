### Installation Termux-Megapackage 
#### Commands for termux
* pkg update && pkg upgrade && pkg install git -y
* git clone https://github.com/Bhavik-Tutorials/Termux-Megapackage
* cd Termux-Megapackage
* chmod +x install.sh
* Now run bash install.sh
