 ### Installation Termux-Megapackage
 * `Commands` for termux

```
$ pkg update && pkg upgrade && pkg install git -y

$ git clone https://github.com/Bhaviktutorials/Termux-Megapackage

$ ls

$ cd Termux-Megapackage

$ ls

$ chmod +x godmode.sh

$ chmod +x metasploit.sh

$ chmod +x tools.sh

$ chmod +x kali.sh

$ bash godmode.sh

$ bash metasploit.sh

$ bash tools.sh

$ bash kali.sh

```
